# Pencil-Eclipse

## Description
A premium online art school platform for mastering hyper-realism, watercolor, and digital art.

## Tech Stack
- React
- Tailwind CSS
- React Router
- Lucide React / SVG Icons

## Features
- Cinematic Hero Carousel
- Responsive Course Catalog
- Individual Course Details with Curriculum Accordion
- Student Masterpiece Gallery
- Optimized for all device sizes
